PC Potion

This website was developed as part of my first-year web development module. The website is built from without the use of any frameworks and is complete with client and server side validation. This is achieved using a range of mixture of JavaScript client-side checks and PHP server side checks.

Users are able to sign up and login to the website, using a MySQL database to store all the details. Passwords are stored securely using an MD5 hash, which encrypts their password.
