Dijkstra Pathfinder

This demo is written using Python, and uses the library Pygame to show the implementation. Dijkstra's Algorithm is for finding the shortest possible path between two points, where the weight of each path is given. This algorithm will keep exploring the shortest available path until the destination is reached, and then display the path it took to get to the destination.

The Pygame application is a visual representation of this algorithm exploring the path is takes using highlights and visual cues. The application will also include a custom written pathfinding algorithm by myself, which the user can switch between when running the program. The program starts by taking the row and width of the grid the user wishes to create, and then it will generate a grid of those proportions with random numbers representating path weights. The start node is in the top-left of the grid, and the destination is in the bottom-right. The user can also edit the weight of each individual cell, this will help show the user how changing values in certain positions may affect the shortest path.
