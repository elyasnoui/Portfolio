Space Raiders

As part of my first-year's module, I was assigned coursework to create a game using a university provided game engine called CityEngine. The game a chose to make was a knockoff version of the original Space Raiders game, all sprites used are not of my creation and were obtained from royalty-free websites.

The game contains 4 levels, which progressively makes the game harder as you progress. The aim of the game is to destroy all the alien spaceships before the timer runs out and obtain the highest score that you can. Coins can be collected to upgrade your spaceships's weapons as well as purchase additional lives. Upgrades can also be refunded in the case where you need more coins to purchase a life. Players are also able to create gamesaves reload into the same state they were in before. Highscores are stored locally, and a leaderboard is available to view in the main menu.
